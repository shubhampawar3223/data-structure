-It is a data structure which follows a particular order in which operations are performed.

*Push:to insert elements.If stack is full then it is called as overflow. 
*Pop:to remove elements.If stack is empty then it is called as underflow.
*Peek or Top:Returns element of top.
*isEmpty: Returns true if stack is empty.

-Time complexity of stack operation is o(1) because no for loop is used here.

Application-
-It is used in balancing of symbols.
-Infix to Postfix 
-Redo-Undo in applications.
-forward and backward feature in browsers.
